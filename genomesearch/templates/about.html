Wheat Genome Searcher API - Documentation

Provided Information

This RESTful API provides complete data on 31,229 individual wheat genes, and partial data on an additional 61,000 genes. All data is categorized by SNPId and includes data on wheat gene cultivars, centimorgan, chromosome position, and official gene name.

Example Query:
http://URL/WRSGGL-Wheat-API-query?SNPId=IWB2330&chrome=1A,7D

How to Make a Query

This single endpoint API uses additional modifiers to queries to narrow search results. The endpoint URL and the additional modifiers are shown on the right, as well as an example query.

  • SNPId= This parameter searches by inclusion, meaning that when using the SNPId modifier all data that includes the unbroken string of input data in its SNPId will be returned.
  • chrome= This parameter requires inputs to be in the form of {number}{letter} where the possible numbers are 1 - 7 and letters are A, B, D with anything outside of these options resulting in an error. Multiple chrome inputs can be input at the same time, but must be separated with a comma.
  • lowercM= Only accepts integers or doubles greater than 0, all else will result in an error.
  • highercM= Only accepts integers or doubles greater than 0, all else will result in an error.

Endpoint URL: http://URL/WRSGGL-Wheat-API-query?

modifiers:

  • SNPId=
  • chrome=
  • lowercM=
  • highercM=
Example Query:
http://URL/WRSGGL-Wheat-API-query?lowercM=10&highercM=15

Error Codes

  • Error 400 - No matching data: From a query that is too specific that results in no data matching the parameters. Try your search again, but with less specificity.
  • Error 421 - Incompatible chrome: From incorrectly formatting a chromosome query. Searching for A1 instead of 1A, or searching for a chromosome that doesn't exist (such as 4C), will result in this error.
  • Error 422 - Incompatible SNPId: From incorrectly formatting a SNIP id query or from searching for a SNIP id that does not exist will result in this error.
  • Error 423 - Incompatible lowercM: From incorrectly formatting a lower bounded cM query or from searching using characters instead of numbers will result in this error.
  • Error 424 - Incompatible highercM: From incorrectly formatting a higher bounded cM query or from searching using characters instead of numbers will result in this error

Example of an erroneous query: http://URL/WRSGGL-Wheat-API-query?SNPId=IWD233&lowercM=low

This results in error 422, Bad Request! Incompatible SNPId. Since error codes are sent based off of the first error encounted only one error code is returned, even though all requested query modifiers are used improperly. To fix this request all modifiers will have to be changed.

Try instead http://URL/WRSGGL-Wheat-API-query?SNPId=IWB233&lowercM=50
